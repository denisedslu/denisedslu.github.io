TweetBeat

TweetBeat is a web app that allows a user to search for trending topics on Twitter within a specified community via user-entered handles. I worked on the prototype with two other computer science students in a collaborative course at Northwestern University.

NBN Year in Media 2013

NorthbyNorthwestern.com's annual Year in Media project is an interactive that recaps the year's biggest headlines. Taking inspiration from The Verge 50, I planned the concept with NBN's Editor-in-Chief then designed the graphics and coded the project.
