iPad magazine app

Samples from an iPad magazine app made using Adobe InDesign CS5 DPS for a Medill media design class final project. The app was for a fictional experimental music magazine I created and included clips I've written in the past as well as original content. Functionalities used included overlay creator slideshow, object states and embedded interactive audio and video clips.
